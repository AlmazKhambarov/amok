Console Mirroring

Console output is written to your command prompt.

Interactive Mode

Use interactive mode to get a remote read-eval-print-loop in your command prompt.

Hot Patching

Active scripts are updated via the runtime debugger as they are saved to disk.

Development Server

Zero configuration development server with automatic index.html generation for the common cases.

Preprocessing

Getting started with Amok

Use npm to install from the command line, then see the Getting Started Guide for more in-depth instructions

. 

      # Install the `amok` command globally
      $ npm install amok

      # Install the `amok` command as a development dependency
      $ npm install amok --save-dev
