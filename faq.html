What editors are supported?

As a command line tool, it works independent of the editor. Currently there are no direct editor integrations.

What frameworks are supported?

There are no direct ties with any frameworks, it should work regardless of the framework.

Why isn't my code updating?

Scripts will not be re-evaluated, that is it will not be executed again because that would cause side-effects to occur, and corrupt the state of the application.

There are events available to further process source changes and apply side effects.
