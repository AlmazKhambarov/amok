What Is It?

Amok is a free, cross-platform, open source standalone command line tool for rapid prototyping, development and debugging of JavaScript web applications.

Amok itself written in JavaScript, runs on node and is distributed via npm

What Does It Do?

Zero Configuration Server

Simple http server, serves scripts and resources, and generates a default index.html response if the file does not exist.

Automatic Preprocessing

Monitors the file system, automatically preprocessing source files through a variety of compilers and bundlers.

Live Code Editing

Upon detection of a loaded script being changed, it gets sourced in the client without interrupting execution, executing side effects or losing state.

Console Mirroring

Output from client's console log, info, error, trace and so on gets mirrored and written to the calling process' stdout and stderr.

Interactive Mode

Interactive mode enables inspection of the client environment as it is running through a read-eval-print-loop.

File Notifications

Notifications are sent to the global object in the client environment, enabling domain specific processing of changes.

What Compiler Can It Use?

Amok can use any compiler that has a command line interface capable of writing to a file with an option to configure the output path.

This is done through a simple command line option or environment variable configuration, which means it can work with CoffeeScript, TypeScript, Babel, Webpack and Browserify just by pointing it to the compiler executable.

How Does It Work?

Amok works by connecting directly to the browser via the remote debugging protocol, which means no browser extension are required.

Using a browser debugging session like this means that it is truly a live editing experience with support from the JavaScript runtime itself, so the application state is never lost, and it can reach deep down into closures without causing side effects.

Where Can I Get It?

Download and install node

Open a terminal window and run the following command

npm install --global amok

How Do I Use It?

amok --client chrome app.js
