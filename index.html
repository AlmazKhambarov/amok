What is it?

Amok is a free, open source standalone command line tool for rapid prototyping and development of JavaScript applications.

Amok is written in JavaScript, runs on node and is distributed via npm

 

What does it do?

Amok monitors changes in the file system. As soon as you save a file, it is then preprocessed/compiled as needed, and reloaded in the browser session without refreshing. This keeps the application state unchanged while doing live edits.

It can also spawn a server for rapid development of front end applications

How does it work?

Amok works by connecting directly to the browser via the debugging protocol, which means no browser extension are required.

Using a browser debugging session like this, means that it is truly a live editing experience with support from the JavaScript runtime itself, so the application state is never lost and it can reach deep down into closures.

 
 

How do i get it?

Prerequisites

Download and install node

Installation

Open a terminal window and run the following command

npm install --global amok
